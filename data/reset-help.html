Password Reset Instructions

Note: This procedure will reset the credentials to the default values:
Username: admin
Password: admin

Hardware Reset Procedure

  1. Locate the credentials reset button
  2. Press and hold the credentials reset button
  3. While holding the credentials reset button, eiter power off/on or press system reset button
  4. Keep holding for 10 seconds until the reset is confirmed
  5. Release the button
  6. The system will reset the credentials to default values
  7. You can now log in with the default credentials

Important: After logging in with default credentials, please immediately change your password through the preferences page for security.
